# Feature Modules

This folder contains the app's feature-owned UI, hooks, services, and context.

Current structure:
- `auth`: authentication pages, auth context, and auth hooks
- `chat`: chat pages, chat UI components, chat hooks, and chat services
- `calls`: one-to-one call UI, hooks, and signaling services
- `profile`: profile pages
- `stories`: story viewer components and story page exports

Shared app-level utilities remain outside feature folders when they are used across multiple domains.
